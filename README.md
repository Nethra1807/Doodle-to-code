# 🎨 Doodle to Code Converter

A machine learning system that converts hand-drawn UI sketches into HTML code.

🚧 Project Status: This project is currently under development.  
Some features are still being implemented and improved.

# 🚧 Current Progress

✔ CNN model trained for UI component classification  
✔ Basic HTML code generation implemented  
✔ Streamlit interface created  

🔄 Currently working on:
- Improving prediction accuracy
- Adding more UI components
- Expanding dataset
